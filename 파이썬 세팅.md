# Python Virtual Environment (venv) Setup

#### 1. Create venv (once)
```
python3 -m venv .venv
```

#### 2. Activate venv
```
source .venv/bin/activate
```
After activation, Python and pip will use the isolated environment under .venv/.

#### 3. Install dependencies
```
pip install -r requirements.txt
```

#### Exit / Deactivate venv
Deactivate venv
```
deactivate
```

This returns the shell to the system Python environment.
